# E-Commerce Sales Data Analysis (SQL)

## Project Overview
This project analyzes e-commerce sales data using SQL to identify revenue trends, top products, customer behavior, and category performance.

## Tools
- SQL (MySQL)

## Key Analysis
- Total revenue and monthly sales
- Top-selling products
- Category-wise revenue
- Customer purchase behavior
- High-value customers

## SQL Concepts Used
SELECT, WHERE, GROUP BY, HAVING, JOIN, Subqueries, CASE WHEN
